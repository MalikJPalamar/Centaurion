# Centaurion Glossary

Key terms and concepts used across the Centaurion exo-cortex system.

## Active Inference Terms

| Term | Definition |
|------|-----------|
| **Active Inference** | Acting on the world to make sensory data match predictions, rather than passively updating models. See [Active Inference Loop](active-inference-loop.md). |
| **Free Energy** | The divergence between a system's generative model and actual sensory input. Minimizing it is the core drive. See [Free Energy Principle](free-energy-principle.md). |
| **Generative Model** | The internal model that generates predictions about the world. In Centaurion: identity files + wiki + memory. |
| **Markov Blanket** | The statistical boundary separating internal from external states. See [Markov Blanket](markov-blanket.md). |
| **Precision** | The confidence-weight assigned to predictions vs. sensory data. See [Precision Ratio](precision-ratio.md). |
| **Prediction Error** | The mismatch between what was expected and what was observed. Drives the COMPARE step. |

## Centaurion System Terms

| Term | Definition |
|------|-----------|
| **Cortex** | The primary reasoning agent. Handles analysis, planning, and complex tasks. |
| **Nova** | The creative and exploratory agent. Handles research, brainstorming, and external sensing. |
| **Daemon** | The autonomous monitoring agent. Handles health checks, alerts, and background tasks. |
| **Exo-cortex** | The complete AI augmentation system — extends cognition beyond biological limits. |
| **Routing Gate** | The decision mechanism that classifies tasks by novelty × stakes × reversibility. See [Routing Gate](routing-gate.md). |
| **Three Laws** | Hierarchy, Routing, Coupling — the governing constraints. See [Three Laws](three-laws.md). |
