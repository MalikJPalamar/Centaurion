# Three Ventures Overview

Malik operates three ventures simultaneously. Centaurion exists to make this sustainable by providing a shared cognitive infrastructure across all three. Each venture is tagged in the system so agents, memory, and workflows route correctly.

## The Ventures

### AOB — Art of Brilliance

- **Domain:** Personal development, facilitator training, corporate workshops
- **Revenue model:** Certification programs, speaking engagements, corporate contracts
- **Current focus:** CRM migration from Ontraport to GoHighLevel, scaling facilitator certification
- **Centaurion role:** Automate admin overhead, track team capacity, surface client insights

### BuilderBee

- **Domain:** AI automation agency — helping businesses implement AI and automation
- **Revenue model:** Client retainers, project-based implementation, GoHighLevel white-label services
- **Current focus:** Client onboarding workflow, AI automation patterns, service productization
- **Centaurion role:** Template client workflows, automate onboarding, monitor project health

### Centaurion.me

- **Domain:** The exo-cortex itself — personal AI infrastructure and methodology
- **Revenue model:** Thought leadership, framework publishing, potential SaaS
- **Current focus:** Building the system (Phases 1-6), documenting methodology as content
- **Centaurion role:** Is the product. Self-referential — Centaurion improves Centaurion.

## Cross-Venture Synergies

| Pattern | From | To | Value |
|---------|------|----|-------|
| Automation playbooks | BuilderBee | AOB | Reuse client automation for internal ops |
| Framework methodology | Centaurion | BuilderBee | Sell the thinking, not just the doing |
| Real-world testing | AOB + BuilderBee | Centaurion | Live ventures validate the system |
| Knowledge transfer | All | All | Insights in one venture accelerate the others |

## Venture Tagging

Every action in Centaurion is tagged by venture: `aob`, `builderbee`, or `centaurion`. This enables:
- Filtered memory recall per venture
- Venture-specific dashboards and health checks
- Cross-venture pattern detection
