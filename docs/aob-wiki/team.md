# AOB Team and Roles

## Structure

AOB operates with a lean team. Malik provides strategic direction while facilitators handle delivery. The Centaurion system automates administrative overhead to keep the team focused on high-value work.

## Roles

### Strategic Leadership
- **Malik Palamar** — Founder, strategy, partnerships, venture integration
  - Sets direction for certification program and corporate offerings
  - Connects AOB methodology with BuilderBee automation capabilities

### Facilitation Delivery
- **Certified Facilitators** — Workshop delivery, client-facing sessions
  - Trained through the AOB certification program
  - Operate semi-independently with playbooks and brand guidelines

### Operations
- **Admin Support** — Scheduling, CRM management, client communications
  - Primary target for Centaurion automation (reduce manual work)
  - CRM migration will simplify tooling requirements

## Capacity Planning

Key constraint: facilitator availability determines delivery capacity. The certification pipeline is the growth lever — more certified facilitators means more concurrent engagements.

## Centaurion Integration

- Daemon monitors team capacity and flags overcommitment
- Automated workflows handle scheduling and follow-up
- Venture-tagged memory ensures AOB context is always available
