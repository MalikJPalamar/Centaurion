## Migration Status

AOB is consolidating from multiple point solutions to a GoHighLevel-centered stack. See [CRM Migration](crm-migration.md) for details.

### Being Replaced
- **Ontraport** → GoHighLevel (CRM, email, landing pages)
- **Manual scheduling** → GHL workflow automation

### Staying
- **Stripe** — Payment processing (integrates with GHL)
- **Zoom** — Workshop delivery platform
- **Google Workspace** — Team collaboration
