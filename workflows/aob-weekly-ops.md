# AOB Weekly Operations Workflow

## Purpose

Structured weekly operations checklist for the Art of Breath venture, covering CRM health, membership management, content scheduling, and facilitator support.

## Weekly Checklist

### 1. CRM Review
- Check Ontraport for new contacts added this week
- Review and resolve any bounced emails or failed sequences
- Verify membership tags are in sync with Mighty Networks access
- Clean up duplicate contacts if flagged

### 2. Membership Management
- Review new signups and cancellations for the week
- Process any pending access requests
- Monitor community engagement metrics (posts, comments, active members)
- Flag any support tickets that need Malik's attention

### 3. Content Pipeline
- Confirm next week's content is scheduled (blog, email, social)
- Review content performance metrics from this week
- Update content calendar if any pieces need to be rescheduled
- Coordinate with facilitators on any guest content submissions

### 4. Facilitator Support
- Review open facilitator support tickets
- Check for upcoming certification renewals (next 30 days)
- Process any new facilitator applications received this week
- Update facilitator directory if any status changes occurred

## Output

Weekly ops summary posted as a GitHub Issue: `[aob-ops] Week of {date}`

```markdown
## CRM Health
- New contacts: X | Bounces resolved: X | Duplicates cleaned: X

## Membership
- New signups: X | Cancellations: X | Net change: +/-X

## Content
- Published: X pieces | Scheduled next week: X pieces

## Facilitator
- Open tickets: X | Renewals due (30d): X | New applications: X
```

## Frequency

Weekly, run as part of the Monday operations cycle.
