## Usage Tips

- Use Midjourney v6 for best results
- Add `--ar` for aspect ratio
- Use `--v 6` for latest model
- For more saturated colors: add `--s 250`
- For less variation: add `--q 2`
- Use negative prompts to remove unwanted elements
